# Inline Dialog

### Compared to Tooltip `Inline Dialog` does take children as props. This is because it's designed to hold more complex components.

## Usage

<Playground>
  <InlineDialog
    placement="bottom"
    width={240}
    height={240}
    trigger={<Button>Button Trigger</Button>}>
    <div
      style={{
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
        height: '100%',
      }}>
      <Text>Wooo! I'm inside a dialog!</Text>
    </div>
  </InlineDialog>
</Playground>

## Focusing

### `Inline Dialog` will automatically focus the first tabbable element in the its body.

<Playground>
  <InlineDialog
    placement="bottom"
    width={240}
    height={240}
    trigger={<Button>Button Trigger</Button>}>
    <div
      style={{
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
        height: '100%',
      }}>
      <Button>Tabbable button</Button>
    </div>
  </InlineDialog>
</Playground>

### If you want to auto focus a specific element instead of the first tabbable element. You have to pass `data-autofocus` to the element.

<Playground>
  <InlineDialog
    placement="bottom"
    width={240}
    height={240}
    trigger={<Button>Button Trigger</Button>}>
    <div
      style={{
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
        flexDirection: 'column',
        height: '100%',
      }}>
      <Button style={{ marginBottom: '8px' }}>Tabbable button</Button>
      <Button data-autofocus>Tabbable button</Button>
    </div>
  </InlineDialog>
</Playground>
