# Select

### `Select` is basically just a wrapper around [react-select](https://github.com/JedWatson/react-select).

### All `react-select` functionality will work as expected, if not please reach out!

### `Select` provides Kenobi theming to `react-select`. The `SelectField` component provides Labels etc.

### For a list of all available `react-select` props go to [Docs](https://react-select.com/props).

## Usage

This is a controlled example using `value` and `onChange`.

<Playground>
  <Component initialState={{ value: '', label: '' }}>
    {({ state, setState }) => (
      <React.Fragment>
        <Select
          name="icecream-flavours"
          value={state}
          options={[
            { value: 'chocolate', label: 'Chocolate' },
            { value: 'strawberry', label: 'Strawberry' },
            { value: 'vanilla', label: 'Vanilla' },
          ]}
          onChange={(e) => setState(e)}
        />
        <div style={{ marginTop: '1rem' }}>
          <Paragraph>
            {state.value
              ? `You have chosen ${state.label}!`
              : 'Please choose a flavour'}
          </Paragraph>
        </div>
      </React.Fragment>
    )}
  </Component>
</Playground>
